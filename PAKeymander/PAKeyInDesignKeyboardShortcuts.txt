If you already have a customized set of keyboard shortcuts that you want to use while typesetting with PA, 
manually assign the keyboard shortcuts below in your customized set.

Product area : Scripts
	Application: PAKey: ClearAllFrames.jsx --- Default: Shift+Ctrl+H, Text: Shift+Ctrl+H
	Application: PAKey: EditText.jsx --- Default: Escape
	Application: PAKey: Extend2Next.jsx --- Text: Shift+Page Down, Default: Shift+Page Down
	Application: PAKey: Extend2Prev.jsx --- Default: Shift+Page Up, Text: Shift+Page Up
	Application: PAKey: GetScopeDefault.jsx --- Default: Shift+Ctrl+Alt+=
	Application: PAKey: GetScopeText.jsx --- Text: Shift+Ctrl+Alt+=
	Application: PAKey: NextCol.jsx --- Default: Page Down
	Application: PAKey: NextPageForAdjust.jsx --- Default: Shift+Ctrl+Alt+-, Text: Shift+Ctrl+Alt+-
	Application: PAKey: NextPara.jsx --- Text: Page Down
	Application: PAKey: PrevCol.jsx --- Default: Page Up
	Application: PAKey: PrevPara.jsx --- Text: Page Up
	Application: PAKey: TogglePageCol.jsx --- Default: Insert, Text: Insert

Product area : Window Menu
	Styles: Character Styles --- Default: Shift+Ctrl+I
	Styles: Paragraph Styles --- Default: Ctrl+I

Otherwise, copy PAKeymander.indk to the "InDesign Shortcut Sets" folder, 
which can be found in a folder named like this:
  %appdata%\Adobe\InDesign\Version [XX.X]\[Lang]\InDesign Shortcut Sets
For example:
  C:\Users\JohnDoe\AppData\Roaming\Adobe\InDesign\Version 17.0\en_US\InDesign Shortcut Sets

After copying the indk file to that folder, go to Edit > Keyboard Shortcuts, and select
the PAKeymander set.
